# Aplikasi Kurir

Aplikasi untuk mengelola pengiriman, dipakai di HP Android maupun iPhone.

## Apa yang bisa dilakukan aplikasi ini

- **Login** — kurir masuk ke aplikasi dengan akunnya
- **Daftar Pesanan** — melihat semua pesanan pengiriman/penjemputan yang masuk
- **Detail Pesanan** — melihat info lengkap satu pesanan (alamat, penerima, dll)
- **Tracking** — melacak posisi pengiriman

## Status saat ini

Aplikasi ini masih versi contoh/latihan. Data pesanan yang muncul masih data
percobaan (belum data asli dari pelanggan).

## Yang masih perlu dikerjakan sebelum bisa dipakai sungguhan

1. **Sambungkan ke data pesanan asli** — saat ini masih pakai data contoh
2. **Sistem login asli** — saat ini login belum diverifikasi ke server
3. **Peta pelacakan asli** — saat ini tampilan peta masih kotak kosong
4. **Notifikasi** — untuk memberi tahu kurir/pelanggan saat status pesanan berubah
